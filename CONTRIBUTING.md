# 🤝 贡献指南

感谢您对 AI Model Comparison 的关注！

## 📝 数据贡献

### 提交新模型数据

请在对应文件中添加模型信息：

1. 在 `data/models-*.md` 中添加模型详情
2. 在 `data/benchmarks.md` 中更新对比数据

### 数据格式

```markdown
## 🏆 模型名称

**定位：** 简要描述

### 基础信息
- **发布厂商：** xxx
- **发布时间：** xxxx年xx月
- **上下文长度：** xxx tokens

### 能力表现
| 能力 | 评分 | 说明 |
|------|------|------|
| 文本生成 | ⭐⭐⭐⭐⭐ | |
```

## ✅ 数据质量标准

1. **来源可靠** - 优先使用官方发布数据
2. **标注来源** - 注明数据出处
3. **注明日期** - 标注数据获取时间
4. **客观公正** - 如实反映模型优缺点

## 🔄 更新频率

- 重要模型发布后 1 周内更新
- Benchmark 数据每月核查
- 价格数据随时更新

## 📋 PR 流程

1. Fork 本仓库
2. 创建新分支
3. 提交数据更新
4. 创建 Pull Request

感谢您的贡献！
