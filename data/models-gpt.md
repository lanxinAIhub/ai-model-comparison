## 🏆 GPT-4o

**定位：** OpenAI 旗舰级多模态模型

### 基础信息
- **发布厂商：** OpenAI
- **发布时间：** 2024年5月
- **参数规模：** 未公开（估计 ~1万亿+）
- **训练数据截止：** 2023年10月

### 能力表现

| 能力 | 评分 | 说明 |
|------|------|------|
| 文本生成 | ⭐⭐⭐⭐⭐ | 顶尖水平，多任务全能 |
| 代码编写 | ⭐⭐⭐⭐⭐ | HumanEval ~90% |
| 数学推理 | ⭐⭐⭐⭐⭐ | GPQA ~53.6% |
| 多模态理解 | ⭐⭐⭐⭐⭐ | 图像/音频/视频全面支持 |
| 中文能力 | ⭐⭐⭐⭐ | 优秀，但非最强 |
| 指令遵循 | ⭐⭐⭐⭐⭐ | 极强，System Prompt 效果好 |

### API 规格
```
上下文长度：128,000 tokens
多模态：✅ 图像、音频、视频输入
输出模式：文本、音频（语音对话）
速率限制：500 RPM / 200,000 TPM
```

### 成本
```
Input：$5.00 / 1M tokens
Output：$15.00 / 1M tokens
```

### 适用场景
- 🌟 复杂推理和多步骤任务
- 🌟 需要图像理解的应用
- 🌟 需要语音对话的助手
- 🌟 高质量内容创作
- 🌟 编程辅助

## 🚀 GPT-4 Turbo

**定位：** GPT-4 的高性价比版本

### 基础信息
- **发布厂商：** OpenAI
- **发布时间：** 2024年4月
- **上下文长度：** 128,000 tokens
- **知识截止：** 2023年12月

### 能力表现
- 比 GPT-4 略强，主要优势在于更新鲜的知识和更低的成本
- 128K 上下文适合长文档处理

### 成本
```
Input：$10.00 / 1M tokens
Output：$30.00 / 1M tokens
```

## 💡 GPT-4o Mini

**定位：** 轻量级多模态模型，高性价比

### 能力特点
- 多模态能力（图像+音频）
- 成本仅为 GPT-4o 的 3%
- 速度更快，适合实时应用
- 适合简单对话和图像识别任务

### 成本
```
Input：$0.15 / 1M tokens
Output：$0.60 / 1M tokens
```
